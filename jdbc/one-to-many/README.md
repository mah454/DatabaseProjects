## One-to-many relationship :

* Create table persons :
```oracle
CREATE TABLE PERSONS (
  ID     NUMBER NOT NULL PRIMARY KEY ,
  NAME   VARCHAR2(20),
  FAMILY VARCHAR2(20)
);
```

* Create table cars : 

```oracle
CREATE TABLE CARS (
  ID   NUMBER NOT NULL UNIQUE ,
  NAME VARCHAR2(20),
  FK_PERSON NUMBER NOT NULL ,
  FOREIGN KEY (FK_PERSON) REFERENCES PERSONS(ID) ON DELETE CASCADE
);
```
